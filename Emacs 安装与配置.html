Emacs 安装与配置

Emacs？

官网翻译过来就是，一个可扩展的、可定制的、免费/自由的文本编辑器——等等。它的强大超乎你的想象。

安装与配置 Emacs

安装

  1. 下载 Emacs，它可以多平台使用，本文记录 Windows10 下安装，点击官网主页面 Windows 下载图标，点击 nearby GNU mirror（自动选择就近服务器下载地址），选择最新版本，no-deps 是没有依赖包的版本。
  2. 解压 Emacs 到 D:/emacs，进入 D:/emacs/bin，右键 runemacs.exe 添加快捷方式到桌面，运行即可使用。

配置

本文使用 redguardtoo 的配置，推荐阅读 一年成为Emacs高手。

Install in normal way

Git 国内速度较慢，建议全局科学上网进行配置且已安装 Git。Emacs 默认家目录为 C:\User\Username\Appdata\Roaming，Win+R 输入 %appdata% 可快速进入此目录（下文提到的 ~ 均为此目录）运行：
git clone https://github.com/redguardtoo/emacs.d.git .emacs.d
clone 完成后，运行 Emacs，提示是否选择中国的最快镜像源，选择 y，耐心等待。当出现 Happy…… 则配置成功。

Install stable version(OPTIONAL)

  1. 下载两个文件 emacs.d-stable.zip 和 myelpa-stable.zip。
  2. 将 emacs.d-stable 解压到 ~/.emacs.d。
  3. 将 myelpa-stable 解压到 ~/projs/myelpa，确保解压的文件里有 archive-contents 文件。
  4. 编辑 ~/.emacs.d/lisp/init-elpa.el，找到 ~/myelpa/ 的所在行，取消注释（删除前面的两个英文分号），将所有的符号 ~ 替换为 C:\User\Username\Appdata\Roaming，共三处。
  5. 运行 Emacs，同上一种安装方法。

个人配置

  • 打开 ~/.emacs.d/init.el，找到 Evil 的所在行，注释此行，Evil 的功能是在 Emacs 下使用 Vim 的快捷键。
  • 将编码环境设置为 UTF-8，习惯统一使用 UTF-8 编码。
  • 禁用出错时系统提示声。
  • 隐藏标题多余 *，只显示一个，更易排版。
  • 使用 文泉驿等宽微米黑 在 Windows 中，中英文显示较均衡。

在 custom-set-variables.el 中添加：

 ;; 将编码环境设置为 UTF-8 格式
(set-language-environment "UTF-8")
 ;; 禁用提示音
(setq visible-bell 0)
 ;; 隐藏多余*
(setq org-startup-indented t)
 ;; 设置字体，在该文件最顶部添加，避免添加自定义主题后设置失效
(when (member "文泉驿等宽微米黑" (font-family-list))
    (set-frame-font "文泉驿等宽微米黑-14" t t))
  • 设置主题

M-x load-theme 或 M-x Customize-themes，习惯用默认主题。
