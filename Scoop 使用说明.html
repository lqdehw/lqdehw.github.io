Scoop 使用说明

Scoop

A command-line installer for Windows，即 Windows 命令行安装程序，或叫 Windows 的包管理器。便于安装各类官方软件，而不用去官网页面下载。

安装

作者说明文档 Scoop，详细视频教程 Scoop Installation Custom Directory。

环境要求

  • Windows 7 SP1+ / Windows Server 2008+。
  • PowerShell 5 and .NET Framework 4.5，若不是安装的时候 Windows 会自动提示下载。
  • 目录不要存在中文。
  • 本文配置是在当前用户下使用 Scoop。

安装过程

  1. 建立目录安装 D:\scoop，默认安装目录是 C:\users\<user>\scoop。
  2. 打开 PowerShell （非管理员）。
  3. 设置当前用户的执行策略为 RemoteSigned（可远程签署），即可执行远程脚本。
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  4. 确定更改策略。 Y
  5. 设置安装目录。$env:SCOOP\='D:\scoop'
  6. 设置用户变量、环境变量，查看用户变量中要存在 D:\SCOOP，环境变量中要存在 D:\scoop\shims。
    [Environment]::SetEnvironmentVariable('SCOOP', $env:SCOOP, 'User')
  7. 执行安装脚本。
    iwr -useb get.scoop.sh | iex

安装成功会显示：Scoop was installed successfully!

使用

  1. scoop help 查看帮助。
  2. scoop bucket list 查看 bucket 清单，默认只有一个 main。
  3. scoop install 7zip 安装 7zip，这个是在 main 中有的软件，安装在 D:\scoop\apps 目录下。
  4. scoop bucket know 查看可用 bucket。
  5. scoop bucket add extras 添加 extras bucket，过程会比较慢，耐心等待。
  6. scoop bucket add dorado dorado 国内的许多软件，QQ、TIM和微信等。
  7. scoop install dorado/<appname> 使用第三方 bucket 安装软件。
  8. Scoop Buckets 查询网站。
