键道6

键道6 是吅吅大牛制作的输入法。特点是“双拼顶功”，数字选重的几率很小，声母和韵母分布在除 viuoa 的 21 个键位上。viuoa 分别对应横（提）、竖（左竖勾）、撇、点（捺）、折（右竖勾）。它以打词为主，当然打单也挺好，习惯使用整句输入的人转到 键道6 要学会合理的判断字词。我在使用 键道6 之前过一段时间的微软双拼，节奏感很强，但由于无法打简拼，且回改很麻烦，最终放弃。 选择它的原因：

  • 几乎不用选重，打词很厉害。
  • 顶功，直接输入后面的词组，前面的词组可以自动上屏，可大大降低空格的敲击次数。
  • 简码，630（105 个声母+笔形 21×5，525 个声母+笔形+笔形 21×5×5）是它的主要特色，可根据自己的需求更换。
  • 每周爱好者会提交词组议表，肥肥 定期更新词库。

具体教程参看 小小文档，这较新。

键道6的安装与配置

Windows安装

Windows 小狼毫键道6
B 站上有详细的 安装教程

配置

  • 自定义词库

打开用户文件夹中 xkjd6.user.dict.yaml，即可添加自己的常用词组。注意：中间一定要使用 Tab 间隔开，这里的 Tab 键是四个空格，重新部署小狼毫即可。但若更新词库后自定义词库会丢失，记得备份到用户数据文件夹，我的是C:\Program Files (x86)\Rime_JD_Tools\Rime_JD\Tools\SystemTools\WindowsTools\用户数据。这里出现下划线_，需在文件顶部添加 #+OPTIONS: ^:nil，否则下划线后面的内容都会变成脚标。也可在配置文件中添加 (setq org-use-sub-superscripts nil) 来进行全局设置。

  • 关闭左右 Shift 切换英文

因 Shift 键在许多软件中都定义了快捷键，容易误触，所以在 WIN10 中我只使用中文一种语言，通过 「Ctrl+空格」关闭当前中文输入法来实现输入英文。在 WIN10 设置中搜索「高级键盘设置」，点击「语言栏选项」即可对 「Ctrl+空格」进行设置。在小狼毫的用户文件夹中 default.custom.yaml，留意代码的缩进，添加：

"ascii_composer/switch_key":
Shift_L: noop   # 关闭左shift切换英文
Shift_R: noop   # 关闭右shift切换英文

重新部署 RIME 即可。

  • 将 Caps-Lock 修改为「左Ctrl」的功能，在桌面新建 Lock.reg，在其中添加：
Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout]
"Scancode Map"=hex:00,00,00,00,00,00,00,00,02,00,00,00,1d,00,3a,00,00,00,00,00

以管理员的方式运行后重启即可生效。

Manjaro安装

基于fcitx-rime,依次执行以下代码

sudo pacman -Rsc fcitx ; 彻底删除 fcitx 包，和依赖 fcitx 包的其它所有包
sudo pacmsn -Rsc ibus ; 彻底删除 ibus 包，和依赖 ibus 包的其它所有包
;; -Rsc 须要谨慎使用，可能一次性删除大量的包，我这里是想完全重新安装才这样操作
;; 删除在 ~/.config 目录下有关 fcitx 和 ibus 的相关配文件的文件夹，资源管理器中 ctrl+h 可以显示隐藏的文件夹，终端 ls -a
;; 重启或注销

sudo pacmsn -S fcitx-rime ; 安装 fcitx-rime
sudo pacmsn -S fcitx-configtool ; 安装后，在配置 fcitx 时会有图形界面

git clone https://github.com/xkinput/Rime_JD.git
cd ~/Rime_JD/Tools/SystemTools/LinuxTools

;;执行，过程中会出现 error,不用管
./2update.sh
;; 最后点击输入法的图标右键-配置-添加中州韵，重启或注销，若未成功再执行一次 ./2update.sh后 重启或注销

;; 添加 ~/.xprofile 环境变量
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"
